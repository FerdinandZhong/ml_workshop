# ml_workshop

## Training

### Environment Setting Up 

* create EC2 instance
* ssh into the instance
* install git by

    ```bash
    sudo apt-get update
    sudo apt-get install -y git
    ``` 

* clone the git repo
* run the start.sh

### Running Training Task

#### install required packages

* source ~/.bashrc
* conda activate workshop-env
* pip install -r requirements_common.txt
